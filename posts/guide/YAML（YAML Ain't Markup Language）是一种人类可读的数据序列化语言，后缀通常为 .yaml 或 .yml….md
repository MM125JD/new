YAML（"YAML Ain't Markup Language"）是一种人类可读的数据序列化语言，后缀通常为 .yaml 或 .yml。它广泛用于编写配置文件、应用程序数据交换以及容器编排（如 Docker 和 Kubernetes）。 [++[1](https://primenumber.com/blog/yaml/)++, ++[2](https://www.runoob.com/w3cnote/yaml-intro.html)++, ++[3](https://github.com/apolloconfig/apollo/issues/4986)++, ++[4](https://www.redhat.com/zh-cn/topics/automation/what-is-yaml)++]  
其核心语法规则包括：  
* **大小写敏感**。 [++[1](https://www.ruanyifeng.com/blog/2016/07/yaml.html)++, ++[2](https://support.huaweicloud.com/devg-cci/cci_05_1001.html)++]  
* **空格缩进**：使用空格（而非 Tab 键）缩进表示层级关系。 [++[1](https://www.ruanyifeng.com/blog/2016/07/yaml.html)++, ++[2](https://support.huaweicloud.com/devg-cci/cci_05_1001.html)++]  
* **注释**：使用 # 开头表示注释。 [++[1](https://www.ruanyifeng.com/blog/2016/07/yaml.html)++, ++[2](https://support.huaweicloud.com/devg-cci/cci_05_1001.html)++]  
* **三种数据结构**：  
    * 对象（键值对）：键值间有空格，嵌套层级对齐。  
    * 数组（列表）：元素前加短横线 - 。  
    * 纯量（标量）：不可再分的值（如字符串、布尔值等）。 [++[1](https://www.ruanyifeng.com/blog/2016/07/yaml.html)++]  
深入了解具体的语法细节和排版规范，可参考 **++[YAML官方网站](https://yaml.org/)++**、**++[菜鸟教程的YAML入门](https://www.runoob.com/w3cnote/yaml-intro.html)++** 或 **++[阮一峰的YAML语言教程](https://www.ruanyifeng.com/blog/2016/07/yaml.html)++**。 [++[1](https://yaml.org/)++, ++[2](https://www.runoob.com/w3cnote/yaml-intro.html)++, ++[3](https://www.ruanyifeng.com/blog/2016/07/yaml.html)++]
