如何创建大纲 ​

打开你的 VitePress 配置文件（比如 .vitepress/config.mts 或 .vitepress/config.js），在 themeConfig 括号内部，添加 outline 配置：

mts
export default {
  title: "我的文档",
  themeConfig: {4,7}
    // ⬇️ 就是添加下面这几行代码
    outline: {
      // level: [1, 6] 表示让右侧目录显示 1 级到 6 级的标题
      // 这样你写的 # 一级标题 就会乖乖出现在右侧了！
      level: [1, 6], 
      
      // 顺便可以把右侧的提示文字改成中文（默认是 On this page）
      label: '本页大纲' 
    }
  }
}

如何改变大纲的位置 ​

在 VitePress 中，右侧的大纲（Outline）作为官方默认主题的一部分，其页面布局位置是固定的（默认在宽屏下悬浮显示在页面右侧，在窄屏/手机端会自动隐藏并折叠进顶部的面包屑菜单中）。

如果你想彻底改变它的视觉位置（比如把它从右侧挪到左侧，或者调整它的上下间距、让它更靠左/更靠右），VitePress 没有提供直接的配置项，但我们可以通过修改自定义全局 CSS 样式来非常灵活地挪动它。

以下是实现改变大纲位置的具体方法：

方法一：微调大纲的上下位置或左右边距（最常见） ​

如果你只是觉得它离内容太近、太远，或者想让它靠上/靠下一点，可以通过调整局部定位来实现。

  1. 打开或创建你项目中的全局样式文件（通常是 .vitepress/theme/custom.css）。

  2. 添加以下 CSS 代码进行微调：

CSS

css
/* 控制右侧大纲容器的位置 */
.VPDocAsideOutline {
  margin-top: 20px;     /* 增大或减小这个值，可以控制大纲整体向上或向下移动 */
  padding-left: 16px;   /* 控制大纲文字距离左侧分割线的距离 */
}

/* 如果想调整包含大纲在内的整个右侧边栏（Aside）的宽度或位置 */
.vp-doc .aside {
  padding-left: 32px;   /* 增大这个值可以让大纲更往右边靠，原理是增加左内边距 */
}

方法二：将大纲从右侧整体挪到左侧 ​

如果你习惯像某些传统博客一样，把页面的目录大纲放在左侧（比如紧贴着左侧的导航栏），可以通过修改 Flex 布局的顺序或者绝对定位来强行改变它的渲染位置。

在 custom.css 中添加以下样式：

CSS

css
@media (min-width: 1280px) {
  /* 1. 改变整个文档大框架的排列顺序 */
  .VPDoc .container {
    display: flex !important;
    flex-direction: row-reverse !important; /* 核心：让右侧边栏和中间主内容反转位置 */
    justify-content: flex-end !important;
  }

  /* 2. 调整原本右侧边栏（现在到了左侧）的边距 */
  .vp-doc .aside {
    order: 1;
    padding-left: 0 !important;
    padding-right: 32px; /* 留出右侧与正文的间距 */
  }

  /* 3. 调整主内容区域的顺序 */
  .vp-doc .content {
    order: 2;
  }
}

💡 怎么让这些 CSS 样式生效？ ​

如果你还没有配置过自定义样式文件，请按照以下步骤让上面的代码生效：

  1. 在 .vitepress/ 文件夹下创建一个 theme 文件夹（如果不存在的话）。

  2. 在 theme 文件夹下创建 custom.css，把上面的 CSS 代码粘进去。

  3. 在 theme 文件夹下创建 index.ts（或 index.js），引入这个样式并导出默认主题：

TypeScript

// .vitepress/theme/index.ts
import DefaultTheme from 'vitepress/theme'
import './custom.css' // 引入你的自定义样式

export default {
  extends: DefaultTheme
}

VitePress 在打包时会自动读取 theme/index.ts，你写在 custom.css 里的位置魔改代码就会直接作用于全站的所有笔记页面了！你是想把大纲调整到左边，还是只是想微调一下它现在的上下间距呢？
